## 1. 启用 Wiki 功能

### 步骤：

1. **访问仓库设置**
   - 打开你的 GitHub 仓库：https://github.com/worldop123/openclaw-gui
   - 点击页面顶部的 **Settings** 标签

2. **启用 Wiki**
   - 在左侧菜单中找到 **Features** 部分
   - 勾选 **Wiki** 复选框
   - 设置会自动保存

3. **验证 Wiki 已启用**
   - 返回到仓库主页面
   - 你应该能在顶部导航栏看到 **Wiki** 标签

---

## 2. 创建首页

### 步骤：

1. **进入 Wiki**
   - 点击仓库顶部的 **Wiki** 标签
   - 点击 **Create the first page** 按钮

2. **设置页面标题**
   - 在 "Page title" 字段中输入：`Home`

3. **复制内容**
   - 打开项目中的 `WIKI_GUIDE.md` 文件
   - 复制该文件的全部内容
   - 粘贴到 Wiki 编辑器中

4. **保存页面**
   - 点击 **Save Page** 按钮
   - 你的 Wiki 首页就创建好了！

---

## 3. 添加更多页面

### 创建新页面：

1. **在 Wiki 首页**
   - 点击右上角的 **New page** 按钮
   - 或者点击页面底部的 **Add a new page** 链接

2. **填写页面信息**
   - **Page title**: 页面标题（例如：`团队介绍`）
   - **内容**: 编写你的页面内容

3. **保存页面**
   - 点击 **Save Page**

### 页面间链接：

在 Wiki 页面中，你可以使用以下语法链接到其他页面：

```markdown
[[团队介绍]] - 链接到 "团队介绍" 页面
[[使用指南|点击这里查看使用指南]] - 带自定义文本的链接
```

---

## 4. 编辑和维护

### 编辑现有页面：

1. **打开页面**
   - 导航到你想编辑的 Wiki 页面
   - 点击右上角的 **Edit** 按钮

2. **进行修改**
   - 在编辑器中修改内容
   - 你可以使用 Markdown 语法

3. **保存更改**
   - （可选）在 "Edit message" 字段中填写修改说明
   - 点击 **Save Page**

### 查看历史版本：

1. **打开页面历史**
   - 在 Wiki 页面上，点击 **Revisions** 标签
   - 你会看到所有历史版本的列表

2. **比较版本**
   - 选择两个版本
   - 点击 **Compare selected revisions**
   - 查看具体的修改内容

3. **恢复旧版本**
   - 找到你想恢复的版本
   - 点击该版本
   - 点击 **Edit**
   - 直接保存即可恢复

## 💡 Wiki 编辑技巧

### Markdown 语法：

GitHub Wiki 支持标准的 Markdown 语法：

```markdown
# 一级标题
## 二级标题
### 三级标题

**粗体文本**
*斜体文本*
~~删除线文本~~

- 无序列表项 1
- 无序列表项 2
  - 嵌套项

1. 有序列表项 1
2. 有序列表项 2

[链接文本](https://example.com)
![图片描述](图片URL)

`行内代码`

```
代码块
```

| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
```

### 有用的技巧：

1. **使用目录**
   - 在长页面开头添加 `[[_TOC_]]` 自动生成目录

2. **添加侧边栏**
   - 创建名为 `_Sidebar` 的特殊页面
   - 它会显示在所有 Wiki 页面的侧边栏

3. **添加页脚**
   - 创建名为 `_Footer` 的特殊页面
   - 它会显示在所有 Wiki 页面的底部

4. **使用图片**
   - 你可以直接将图片拖拽到编辑器中
   - 或者使用外部图片链接
